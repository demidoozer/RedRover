# RedRover
A coding exercise to demonstrate competence and ability. 
# Details
This is a console application written in net10.0. The expected parameters are an alphabetical flag and the string to be parsed. The flag can be omitted. Once entered the string will be parsed and displayed in the console.
# Testing
From the project root run ```bin/Release/net10.0/RedRover -a "(id, name, email, type ( id, name test, customFields(c1, c2, c3)), externalId)"```
